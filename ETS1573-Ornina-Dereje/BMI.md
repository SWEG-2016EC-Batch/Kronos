BMI Calculation Algorithm

1.Input:

 - Weight (in kilogram)
 - Height (in meter)

2.Process:
-Calculate BMI using the formula: 

BMI = Weight/(Height^2)

3.Output:

-print the BMI value & prompt for continuation

-Determine the BMI category:

    -Underweight: BMI < 18.5
    -Normal weight: 18.5 <= BMI < 24.5
    -Overweight: 25 <= BMI <=29.5
