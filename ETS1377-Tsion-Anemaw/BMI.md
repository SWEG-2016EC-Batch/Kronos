 BMI Calculation Algorithm

1. Input: 
 
        - Weight (in kilogram)
        - Height (in meter)

3. Process:
    - Calculate BMI using the formula: 
       BMI = Weight/(Height^2)
      

 4. Output:
    - Display the BMI value
    - Determine the BMI category:
      - Underweight: BMI < 18.5
      - Normal weight: 18.5 <= BMI < 24.9
      - Overweight: 25 <= BMI <=29.5
